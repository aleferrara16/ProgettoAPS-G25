# Progetto APS : Sistema di E-Voting per Referendum Universitario
> **WP1, WP2 & WP3: Specifica del Modello, Architettura e Analisi di Sicurezza**

Questo repository contiene l'implementazione prototipale del sistema di voto elettronico (E-Voting) centralizzato ma crittograficamente verificabile, progettato per le consultazioni referendarie universitarie.

L'obiettivo principale del sistema è disaccoppiare matematicamente l'identità civile del votante (gestita dall'Ateneo) dalla preferenza espressa (gestita dall'Urna), garantendo **Anonimato**, **Unicità del voto**, **Resilienza** e **Receipt-Freeness**.
